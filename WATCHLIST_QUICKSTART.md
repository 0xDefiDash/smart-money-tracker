## Get Instant Alerts When Wallets Make Trades

### Step 1: Add a Wallet to Your Watchlist

1. **Navigate to Wallet Tracker**
   - Click "Wallet Tracker" in the sidebar
   - Or visit: https://defidashtracker.com/wallet-tracker

2. **Add Wallet Address**
   - Click "Add to Watchlist" button
   - Enter wallet address (e.g., `0x742d35Cc6634C0532925a3b844Bc9e7595f0bEb`)
   - Select blockchain (Ethereum, Base, BNB, Solana, etc.)
   - Add label (optional): "Whale #1" or "Smart Money Wallet"
   - Click "Save"

3. **Optional: Track Specific Token**
   - Enter token contract address
   - Enter token symbol (e.g., USDC)
   - Only alerts for this token will be sent

### Step 2: Enable Telegram Notifications (Optional)

1. **Open Telegram**
   - Search for: `@YourDeFiDashBot`
   - Click "START" button

2. **Link Account**
   - Bot will send your chat ID
   - Go to Settings in DeFiDash
   - Enter your Telegram username and chat ID
   - Save settings

### Step 3: Start Receiving Alerts

**That's it!** The system now monitors your watchlisted wallets automatically.
